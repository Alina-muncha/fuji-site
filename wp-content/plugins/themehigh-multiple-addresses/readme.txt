To access the plugin, 

Go to Dashboard ⟶Navigate to WooCommerce⟶Manage Address.

== Installation ==
= Minimum Requirements =
* WooCommerce 3.0 or greater
* WordPress 5.2 or higher

= Automatic installation =
1. Log in to your WordPress dashboard.
2. Navigate to the Plugins menu, and click "Add New".
3. Search and locate 'Woocommerce Multiple Addresses' plugin.
4. Click 'Install Now', and WordPress will take it from there.

= Manual installation =
Manual installation method requires downloading the 'Woocommerce Multiple Addresses' plugin and uploading it to your web server via your FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).

== Frequently Asked Questions ==
= How to enable Multiple Address for Billing and Shipping? =
<ol>
<li>Go to Dashboard⟶Navigate to WooCommerce⟶Manage Address⟶General Settings tab.</li>
<li>Select the Enable Multiple Address for Billing checkbox to enable multiple address for billing.</li>
<li>Select the Enable Multiple Address for Shipping checkbox to enable multiple address for shipping.</li>
<li>Click Save Changes to confirm the changes.</li>
</ol>

= How to view the saved addresses?  =
<ol>
<li>Go to My Account Page⟶Navigate to Addresses tab</li>
<li>You can view the default Billing and Shipping addresses as well as the additional Billing and Shipping addresses.</li>
</ol>

= How to enable Billing Address Autofill? =
<ol>
<li>At the Checkout Page, upon clicking the Billing Details dropdown menu, the saved addresses will be displayed. Also, Add New Address option will also be displayed.</li>
<li>When you click any of the desired Address, all of the fields will be auto-filled.</li>
<li>When you click on Add New Address, you can add a new address from there itself.</li>
</ol>
